# Weather (iOS Version)

- Simple iOS App for displaying week weather.

- Topics:
  * Clean Architecture: 3 Layers : (Presentation, Domain, Data) Layers.
  * Presentation Layer:
    * SwiftUI.
    * View-Model.
    * MVVM Presentation Architecture Pattern.
  * Domain Layer:
    * UseCases.
    * Repository Interfaces.
  * Data Layer:
    * Repository Implementation.
    * Models.
    * APIs: URL Session.
